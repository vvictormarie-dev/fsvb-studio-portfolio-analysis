# Reference

Ce dossier centralise les explications stables sur le fonctionnement du depot.

Il sert a garder sous la main :

- la logique generale du projet
- le role des dossiers principaux
- le role des agents Codex specialises
- les nouveaux angles d'audit comme l'accessibilite, le SEO et la securite
- les points utiles pour une personne peu technique

Documents disponibles :

- `repository-overview.md` : vue d'ensemble du depot et de son organisation
- `agents-guide.md` : role de chaque agent et cas d'usage simples
- `glossaire-debutant.md` : definitions simples des mots utiles du projet

Conseil d'usage :

- lire ce dossier pour comprendre "comment le projet est pense"
- lire `docs/how-to/` pour savoir "comment faire"
- lire `reports/` pour voir "ce qui a ete observe"
